La fameuse attestation

Qui a déjà oublié de faire une attestation pour faire ces courses ou même pour se promener ? J'ai pu m'appercevoir que certaines personnes ont tendance a oublié de faire leur attestation avant de sortir de chez eux car c'est "trop long".

Je me suis donc penché sur ce problème et j'ai tenté de le résoudre. Grâce à mes connaissances acquises durant mon DUT Informatique j'ai directement su par où commencer, comment commencer etc ...

J'ai finalement utilisé le langage Python ainsi qu'une librairie qui se nomme Selenium.
Pour faire simple, Selenium est un outil d'automatisation de navigateur web. Il permet d'écrire des scripts dont l'exécution réalisera automatiquement des actions dans un navigateur web comme par exemple : visiter une page, cliquer sur un lien, remplir un formulaire, etc ...

Ce script rempli automatiquement les champs sur le site du ministère et télécharges le PDF. Un gain de temps considérable.

"N'essaie pas de tout faire.
Fais une chose bien."

Steve Jobs

C'est avec cette citation de Steve Jobs que je conclus sur ce projet personnel qui m'a permis de découvrir des tas d'outils, modules mais aussi, ce projet m'a apprit à me perfectionner.

Ce projet est disponible dans un dépôt Github !
